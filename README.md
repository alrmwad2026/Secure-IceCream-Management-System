# Secure Ice Cream Management System

## Project Description (وصف المشروع)

A secure web-based Ice Cream Management System developed using PHP and MySQL.

نظام ويب لإدارة متجر الآيسكريم، يتيح للمدير إدارة المنتجات والبيانات من خلال لوحة تحكم، مع تطبيق مبادئ Secure Software Development Life Cycle (S-SDLC) أثناء تطوير المشروع.


## Technologies Used (التقنيات المستخدمة)

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

تم استخدام هذه التقنيات لبناء واجهة النظام، تنفيذ العمليات البرمجية، وإدارة قاعدة البيانات

## Main Features (الوظائف الرئيسية)

- Admin Login (تسجيل دخول المدير)
- Add Products (إضافة المنتجات)
- Update Products (تعديل المنتجات)
- Delete Products (حذف المنتجات)
- Display Products (عرض المنتجات)
- Upload Product Images (رفع صور المنتجات)


## Security Considerations (الجوانب الأمنية)

- Input Validation (التحقق من صحة المدخلات)
- Authentication Protection (حماية تسجيل الدخول)
- Session Management (إدارة الجلسات)
- Protection Against Common Web Attacks (الحماية من الهجمات الشائعة)


## S-SDLC Phases Applied (مراحل دورة التطوير الآمن)

1. Planning (التخطيط)
2. Requirements (تحليل المتطلبات)
3. Design (التصميم)
4. Development (التطوير)
5. Testing (الاختبار)
6. Deployment (النشر)
7. Maintenance (الصيانة)


## How to Run (طريقة التشغيل)

1. Install XAMPP.
2. Copy the project folder into htdocs.
3. Import ice_cream.sql into MySQL.
4. Start Apache and MySQL.
5. Open the project in the browser.


تم تطوير هذا المشروع كتطبيق عملي لمراحل S-SDLC في تطوير تطبيقات الويب الآمنة.
